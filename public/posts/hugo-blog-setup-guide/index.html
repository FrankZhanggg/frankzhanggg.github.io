前言

最近我决定搭建一个个人博客来记录和分享我的学习和思考。在研究了多种博客解决方案后，我选择了Hugo作为我的静态网站生成器，并使用了Ladder主题。本文将记录我从零开始搭建博客的完整过程，希望对有类似需求的朋友有所帮助。

Hugo简介

Hugo是一个用Go语言编写的静态网站生成器，以其速度快、配置简单和灵活性高而闻名。与WordPress等动态博客系统相比，Hugo生成的是纯静态HTML文件，不需要数据库，加载速度更快，安全性更高，且易于部署。

Ladder主题介绍

Ladder是一个简洁、美观的Hugo主题，中文即阶梯的含义，寓意着读书为输入，写作为输出，都是获取知识的阶梯。该主题支持暗黑模式、响应式设计、多语言支持等特性，非常适合个人博客使用。

安装Hugo

首先，我们需要安装Hugo。在macOS上，可以使用Homebrew来安装：

brew install hugo

安装完成后，可以通过以下命令验证安装是否成功：

hugo version

创建新的Hugo站点

安装好Hugo后，我们可以创建一个新的Hugo站点：

hugo new site blog
cd blog

或者如果你想在当前目录创建：

hugo new site . --force

安装Ladder主题

Hugo支持主题系统，我们可以使用git submodule来安装Ladder主题：

git init
git submodule add https://github.com/guangzhengli/hugo-theme-ladder.git themes/ladder

Ladder主题的仓库地址是：https://github.com/guangzhengli/hugo-theme-ladder

创建内容

现在我们可以开始创建博客内容了。首先，让我们创建一篇示例文章：

hugo new content posts/my-first-post.md

编辑这篇文章，添加一些内容：

+++
date = '2025-03-13T15:38:21+08:00'
draft = false
title = '我的第一篇博客文章'
tags = ['Hugo', '博客']
categories = ['教程']
+++

## 欢迎来到我的博客

这是我使用Hugo和Ladder主题创建的第一篇博客文章。Hugo是一个非常快速的静态网站生成器，而Ladder是一个简洁美观的主题。

2. **易于使用** - 简单的配置和内容管理
3. **灵活性** - 支持多种内容格式和自定义主题

最后，我们需要创建一个归档页面：

hugo new content archives/_index.md

编辑归档页面：

---
title: "文章归档"
layout: "archives"
url: "/archives"
summary: "archives"
---

部署博客

完成本地开发后，你可以使用以下命令生成静态网站文件：

hugo

生成的静态网站将位于 public 目录中。你可以将这些文件部署到任何静态网站托管服务，如GitHub Pages、Netlify、Vercel等。
